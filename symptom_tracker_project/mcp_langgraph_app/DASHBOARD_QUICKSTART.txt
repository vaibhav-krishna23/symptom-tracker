┌──────────────────────────────────────────────────────────────────────────────┐
│ 🚀 INSTALLATION (30 seconds)                                                 │
└──────────────────────────────────────────────────────────────────────────────┘

1. Install dependency:
   cd symptom_tracker_project/mcp_langgraph_app
   pip install pandas

2. Start backend:
   uvicorn api.main:app --reload --host 0.0.0.0 --port 8000

3. Start frontend:
   streamlit run streamlit_app/app_v2.py

4. Access: http://localhost:8501 → Login → Click "📊 Dashboard"

┌──────────────────────────────────────────────────────────────────────────────┐

✅ Average Intensity Analysis (Bar chart by symptom)
✅ Recent Sessions (Last 5 with color coding)
✅ AI Health Insights (Personalized recommendations)

⚠️  ORANGE = Warning (Severity 6-7)  → Monitor closely
✅ GREEN  = Normal (Severity < 6)    → Routine monitoring

│ 🐛 TROUBLESHOOTING                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

Issue: "No module named 'pandas'"
Fix:   pip install pandas

Issue: Charts not showing
Fix:   Restart Streamlit, check browser console

Issue: "No symptom logs yet"
Fix:   Log symptoms first via "🌡️ Log Symptoms"

Issue: Empty dashboard
Fix:   Ensure you're logged in and have logged symptoms
